# Multi-Feature Anomaly Detection System

## Overview
This project presents a multi-feature anomaly detection system using unsupervised machine learning.

## Problem
In modern systems, anomalies often indicate fraud, inefficiencies, or abnormal behavior patterns. Detecting these anomalies is essential for maintaining system reliability and trust.

## Approach
- Simulated multi-feature dataset (amount, frequency, time gap)
- Applied Isolation Forest for anomaly detection
- Visualized anomaly patterns for interpretability

## Features Used
- Transaction Amount
- Transaction Frequency
- Time Gap

## Model
Isolation Forest (Unsupervised Learning)

## Key Insight
Using multiple behavioral features significantly improves anomaly detection compared to single-variable methods.
